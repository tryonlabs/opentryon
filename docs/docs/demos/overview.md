# Demos

OpenTryOn includes interactive Gradio demos for easy experimentation.

## Available Demos

### Extract Garment Demo

```bash
python run_demo.py --name extract_garment
```

### Model Swap Demo

```bash
python run_demo.py --name model_swap
```

### Outfit Generator Demo

```bash
python run_demo.py --name outfit_generator
```

Each demo launches a web interface for interactive use.
