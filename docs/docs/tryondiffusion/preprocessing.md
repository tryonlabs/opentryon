# TryOnDiffusion Preprocessing

Preprocessing pipeline for TryOnDiffusion training data.

## Required Data

- `ip`: Target person images
- `jp`: Person pose embeddings
- `jg`: Garment pose embeddings
- `ia`: Cloth-agnostic RGB
- `ic`: Segmented garments

See [TryOnDiffusion README](https://github.com/tryonlabs/opentryon/tree/main/tryondiffusion/README.md) for preprocessing details.
