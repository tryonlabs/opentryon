# OpenTryOn Documentation

This directory contains the Docusaurus documentation site for OpenTryOn.

## Development

```bash
cd docs
npm install
npm start
```

This starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

## Deployment

The site can be deployed to GitHub Pages, Netlify, Vercel, or any static hosting service.

See the [Docusaurus documentation](https://docusaurus.io/docs) for more details.
